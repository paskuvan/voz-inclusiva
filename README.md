Convierte tus **palabras en voz** para crear narraciones en **videos, reels y contenido digital** de forma rápida y accesible.

**Voz Inclusiva** es un proyecto que transforma texto en audio utilizando tecnologías de **Text-to-Speech (TTS)** para facilitar la creación de contenido audiovisual.

---

## 🚀 Características

- ✨ Conversión de **texto a voz**
- 🎧 Generación automática de **audio narrado**
- 📱 Ideal para **reels, shorts y contenido social**
- ⚡ Flujo simple para generar narraciones
- ♿ Enfoque en **accesibilidad e inclusión**

---

## 🛠️ Tecnologías

Este proyecto utiliza tecnologías como:

- Python  
- APIs de **Text-to-Speech (TTS)**  
- Generación de audio automatizada

## 💡 Objetivo del proyecto

El objetivo de Voz Inclusiva es facilitar la creación de contenido audiovisual accesible, permitiendo transformar texto en voz de forma simple y rápida.

También busca explorar cómo la inteligencia artificial puede mejorar la accesibilidad digital.
